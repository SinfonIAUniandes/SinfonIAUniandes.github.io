"Customer service with facial recognition"

  • 2019

This project is carried out by Adelaida Zuluaga, in which the aim is for the robot to be able to identify and recognize users based on facial recognition techniques, whose execution is local. For this, the detection of people was carried out using the Viola-Jones algorithm, achieving an accuracy of 98% in the visual field of the robot. For the recognition of the person from the detected face, the Eigenfaces algorithm is applied, where the classification is carried out with SVMs one vs all, where the classes correspond to the names of the people. It should be noted that the complete algorithm for the recognition of people enables the training of new classes, which allows the robot to meet new people during its interaction and remember them later. One of the major contributions of this project is that it provides the robot with local capabilities for the identification and recognition of people, serving as a backup method in the event of cloud service failures.

"Opera learns to play line 4 through Reinforcement Learning"

  • 2019

This project is carried out by Juan José García. In it, the aim is for the opera robot to learn to create game strategies that allow it to function in two-player board games. For this, the line 4 game was used as the test framework. To learn from experience, reinforcement learning was applied, specifically, the Q-learning algorithm with neural network. Opponents were designed, varying the level of complexity of their plays, based on game theory, with methods such as expectimax and minimax. With these opponents, Opera's performance playing line 4 is expected to be based on predicting the opponent's behavior a given number of moves forward.

University space mapping with Humanoid Robot Pepper

  • 2021

Currently, one of the most studied fields in robotics is navigation, in particular the ability of a robot to generate a map of a given environment and locate in it. This project seeks to understand the operation of the library RTAB-Map (Real Time Appearance Based Mapping) of ROS, which uses a graph-based SLAM approach, with the purpose of mapping a static scene using the humanoid robot Pepper. With this purpose in mind some of the parameters of this library were studied and documented and They did different tests, both in a controlled environment and in a space free, to determine how it infers the value of those parameters in the maps generated by this library.
